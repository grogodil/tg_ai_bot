# tg-ai-bot

## Описание проекта

Этот проект представляет telegram-бота, который фыполняет 3 функции:
- выдает информацию о себе
- проводит викторину из 3-х простых вопросов с проверкой правильности ответов
- дает совет от ИИ по соответствующему запросу

## Инструкция по запуску кода локально
1. Убедитесь, что у вас установлен Go версии 1.23 или выше.
2. Установите зависимости:
   ```bash
   go mod tidy
   ```
3. Создайте файл `.env` в корне проекта и добавьте в него следующие переменные:
   ```
   TELEGRAM_BOT_TOKEN=токе_для_вашего_бота (его нужно получить в Telegram у @BotFather)
   OPENROUTER_API_KEY=вводите свой ИИ_API_ключ
   ```
4. По умолчанию используется модель ИИ "nvidia/nemotron-nano-9b-v2:free". Если хочется исаользовать другую, то в файле /internal/ai/client.go в коде функции "Ask" в переменной "body" еужно заменить название модели на свое напротив поля "Model"
5. Запустите Telegram-бота:
   ```bash
   go run main.go
   ```
5. Откройте диалог с ботом, введите /start.
